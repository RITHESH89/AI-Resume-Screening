# AI Resume Screening System

This project is a simple AI-based Resume Screening System built using Python and Streamlit.

## Features
- Upload Resume (PDF)
- Enter Job Description
- Calculates Match Score using NLP (TF-IDF)
- Displays similarity percentage

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- NLP

## How to Run

pip install -r requirements.txt
streamlit run app.py

---


This project demonstrates basic AI and NLP concepts for resume-job matching.
